# Segurança (SECURITY)

## Política de Segurança

Valorizamos muito a segurança de nossa plataforma. Se você descobrir uma vulnerabilidade de segurança no RZND Media, solicitamos que nos envie um relatório de forma responsável.

### Reportando uma Vulnerabilidade

*As instruções de contato seguro, criptografia de chaves PGP (se houver) e processos de triagem/divulgação serão preenchidos aqui.*

Por favor, não abra uma issue pública no GitHub para reportar uma vulnerabilidade de segurança.
