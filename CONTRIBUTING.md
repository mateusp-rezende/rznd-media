# Guia de Contribuição (CONTRIBUTING)

Agradecemos o seu interesse em contribuir para o projeto **RZND Media**! Como um projeto Open Source, as contribuições da comunidade são o que tornam esta plataforma excelente.

## Como Posso Contribuir?

*As diretrizes e regras detalhadas para submissão de Issues, Pull Requests, padrões de código e fluxo de desenvolvimento serão inseridas aqui.*

### Padrão de commits
- Recomendamos o uso de [Conventional Commits](https://www.conventionalcommits.org/).

### Submetendo um Pull Request
1. Faça um Fork do projeto.
2. Crie uma branch para sua modificação (`git checkout -b feature/minha-feature`).
3. Faça commit de suas alterações (`git commit -m 'feat: adiciona nova funcionalidade'`).
4. Envie a branch (`git push origin feature/minha-feature`).
5. Abra um Pull Request.
